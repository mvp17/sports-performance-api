Conditions of use

CSV files
COLUMN NAMES in both file types must NOT contain ACCENTS.
The format for saving the files must be of the type "CSV (delimited by commas) (* .csv)".
The event file must have the name of the millisecond column as "TMilisegundos", the column name of the millisecond duration as "DuracionMiliseg". Both column names with NO WHITE SPACES.
The device file must have the time column name as "TIME". That column name with NO WHITE SPACES.
Through the web
Load EVENT FILE FIRST in case TIME series of DEVICE FILES has to START with the FIRST time VALUE of EVENT file. WITHOUT event file first loaded (first in file list), device files' TIME series will START by 0.
